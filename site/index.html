Darkmesh

A sovereign data layer for agent collaboration: private signals, not centralized raw data.

Nodes coordinate on constrained signals while personal context stays local. Discovery and identity reveal are separate, consent-gated steps.

  • Local-first data on each operator’s node
  • Privacy-preserving coordination across peers
  • Optional Neotoma fork integration as the local graph substrate
